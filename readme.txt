Horizontal scroll image slideshow lets you showcase images in a horizontal scroll like fashion, one image at a time and in a continuous manner, with no breaks between the first and last image.

= Features of this plugin =
 
* Simple
* Easy installation.  
* Widgets, so you can add pretty much anything.  
* Easy slideshow size (width & height) override option.
